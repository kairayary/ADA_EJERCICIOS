La Programación Orientada a Objetos (POO) es un paradigma de programación que se 
centra en la creación de objetos y clases para resolver problemas. Cada objeto representa una entidad con propiedades y comportamientos definidos, 
lo que facilita la modelización de problemas del mundo real.
Los principales beneficios de la POO en comparación con la programación estructurada son:
Reutilización de código, Encapsulación, Herencia, Polimorfismo.
Algunos lenguajes de programación que soportan POO son:
Java
C+
Python
C#
JavaScript (con clases y objetos)
